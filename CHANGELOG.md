# Changelog

All notable changes to this theme template will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-11

### Added
- Initial release of Tintero Theme Template
- Complete CSS variable documentation
- Theme metadata JSON structure
- Comprehensive README with examples
- Preview image placeholder
- MIT License
- Contributing guidelines
- Example theme variations

### Features
- All available CSS variables documented with descriptions
- Support for custom CSS rules beyond variables
- Organized variable sections (fonts, colors, components)
- Theme testing guidelines
- Publishing instructions

## Template Usage

When you create your own theme, maintain a changelog like this:

```markdown
## [1.1.0] - 2024-XX-XX

### Added
- New accent color variations
- Custom scrollbar styles

### Changed
- Updated primary background color for better contrast
- Increased default font size

### Fixed
- Border color now properly inherits from variables
```
